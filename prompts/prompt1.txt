 PROMPT 1
Tengo un juego hecho en HTML, CSS y JavaScript llamado "Cazando"
(un cuadrado que atrapa comida dentro de un canvas). Mejora SOLO la
interfaz gráfica modificando únicamente los archivos HTML y CSS, sin
tocar el JavaScript. No cambies los id de los elementos (areaJuego,
puntos, tiempo, mensaje), ni los onclick de los botones, ni las
etiquetas <script>, ni el tamaño del canvas (500x500). Quiero un
estilo temático de Viejo Oeste, como un cartel de "SE BUSCA".
Te paso el cazando.html y el cazando.css.

PROMPT 2

Oeste: una tipo cartel western para el título
